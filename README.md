# How to use

>Please ensure that you run the `npm run prettier:check` and `npm run prettier` to write best code quality before push the code.

### Set up the project

`npm install`

### Running the project

`npm run start`

### Testing the project

`npm run test`

### Running ESLint

`npm run lint`

### Running code format check

`npm run prettier:check`

### Formating your code

`npm run prettier`
